## **🎨 Design System Requirements**

- Surface: #F5F5F5 (Light Gray)

- Text Secondary: #757575 (Medium Gray)

### **Typography**
- **Font Family**: Inter, Roboto (or similar modern sans-serif)
- **Heading 1**: 32px, Bold (700), Line height 1.2
- **Heading 2**: 24px, Bold (700), Line height 1.3
- **Heading 3**: 20px, Semi-bold (600), Line height 1.4
- **Body Large**: 16px, Regular (400), Line height 1.5
- **Body Medium**: 14px, Regular (400), Line height 1.5
- **Body Small**: 12px, Regular (400), Line height 1.4
- **Caption**: 12px, Regular (400), Line height 1.3

### **Spacing (8px Grid System)**

- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

### **Border Radius**
- Small: 4px
- Medium: 8px
- Large: 12px
- Extra Large: 16px
- Full: 9999px (for circular elements)

- Elevation 1: 0px 2px 4px rgba(0,0,0,0.1)

## **✨ Interaction & Animation Requirements**

### **Transitions**
- Page transitions: Fade in/out (300ms)
- Slide transitions: Slide from right (250ms)
- Modal appearance: Scale up (200ms)
- Navigation: Fade with slight scale (200ms)

### **Button States**
- Default: Full opacity, no shadow
- Hover: Slight elevation (shadow)
- Pressed: Color slightly darker
- Disabled: 50% opacity, no interactions

### **Form Validations**
- Empty: Show placeholder
- Focused: Highlight border with primary color
- Valid: Green checkmark icon
- Invalid: Red error message below
- Error: Shake animation

### **Loading States**
- Spinner animation (circular)
- Skeleton screens for content loading
- Progress bar for file uploads
- Pulse animation for loading cards

### **Error States**
- Error icon (red)
- Error message in red text
- Shake animation on entry
- Retry button

### **Success States**
- Checkmark animation
- Green color
- Success message
- Optional confetti animation
